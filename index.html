Hello World

By writing Hello world as a first program for a person is a traditional way to start a new programming language.

Why Hello World?

When a computer or electrical device says hello world, it's like a machine from virtual world greeting the physical world by saying "hello world".
That's why we make output of our first program as "hello world".

My Experience

I also started all of my new programming languages by writing hello world program at the first and it is always been a nice experience to start a new programming language with it because hello world has always been a very easy program for me to begin a new programming language with it.This program doesn't need any variable declaration or input opperation. Hence I think hello world is one of the most preferable program for a beginner to start with.
